# 画像解像度アップ - クイックスタート

## 完全無料・インストール不要

### 1. 画像をダウンロード
```
スライム: https://raw.githubusercontent.com/mameogalaxy/sukuwatto/main/Generated%20Image%20November%2008,%202025%20-%205_49PM.png
パンプキング: https://raw.githubusercontent.com/mameogalaxy/sukuwatto/main/Generated%20Image%20November%2008%2C%202025%20-%206_23PM.png
```

### 2. 画像を保存
`images\original\` フォルダに保存

### 3. 実行
`upscale-simple.bat` をダブルクリック

### 4. 完成
`images\upscaled\` に4倍解像度の画像が生成される

**完全無料** - Windowsの標準機能のみ使用
