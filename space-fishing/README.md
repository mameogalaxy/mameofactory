# 🎣🪐 STELLAR ANGLER 〜伝説のヌシを求めて〜

デュアルショック2のアナログスティックをフルに使った **3D釣りアクション**。
いろんな惑星を飛び回り、宇宙魚を釣りまくり、伝説のヌシ「レヴィアコス」を追え！

ブラウザだけで動く HTML5 / Canvas 製ゲーム（追加インストール不要）。

## 遊び方

`space-fishing/index.html` をブラウザで開くだけ（外部ファイル・追加インストール不要）。
**スマホ／PC／ゲームパッドすべて対応**。

### 📱 スマホ：シームレスなタッチ操作（ボタン無し）

画面を直接さわって操作します。

| 場面 | 操作 |
|------|------|
| タイトル | タップで開始 |
| 惑星選択 | スワイプで切替・中央タップで出発 |
| キャスト | 海面を**なぞって狙い、指を離して**投擲 |
| フッキング | アタリの「！」で**タップ** |
| ファイト | 画面を**なぞって巻く＆引きに逆らう** / **タップで攻撃・エレキ** |

> なぞる速さ＝リールの速さ。魚が走る向きと**逆向きになぞる**とテンションが下がりロッドパワーが溜まる。
> ロッドパワー or エレキが満タンのときに**タップ**すると攻撃が発動（エレキ優先）。

### 🎮 ゲームパッド（DualShock2 想定）／⌨️ キーボード

| 操作 | パッド | キーボード |
|------|--------|-----------|
| 狙い / 竿さばき | 左スティック | W A S D |
| リール | 右スティック（回す） | 方向キー |
| 決定 / キャスト / ロッド攻撃 | ○ | Space |
| エレキアタック | △ | E |
| 戻る | × | Z |

## ゲームの流れ（コアループ）

1. **キャスト** — 左スティックで海面を狙い、○ を溜めて宇宙魚のポイントへ投げる。
2. **アタリ → フッキング** — 宇宙魚がルアーに食いついたら「！」の瞬間に ○ で合わせる。
3. **ファイト** — ここが本番！
   - 右スティックを**回してリール**を巻き、距離を詰める（LANDING ゲージ）。
   - 魚が暴れる「RUN」中に巻くと **テンション**が急上昇。左スティックで**引きに逆らって**竿さばきすると、テンションが下がり **ロッドパワー**が溜まる。
   - **ロッドパワー満タン → ○ でアタック**。宇宙魚のスタミナを削る。
   - テンションが危険域＝**ピンチ！** ここで急速チャージされる **エレキゲージが満タンになったら △ でエレキアタック**＝大ダメージ＋ピンチ脱出。
   - スタミナを 0 にして **KO** → 一気に巻き上げて **GET!**
   - テンションが振り切れると **ライン破断**＝逃げられる。
4. 各惑星で規定数を釣ると **PLANET CLEAR**、次の惑星が解放。
5. 最果ての惑星で **伝説のヌシ** を釣り上げればクリア！

## 惑星と宇宙魚

| 惑星 | 難易度 | 主な宇宙魚 |
|------|--------|-----------|
| ネオン水星 | ★ | グッピスター / コメットコッド |
| 溶岩のヴァルカン | ★★ | マグマレイ / パイロバス |
| 氷結のグラシエ | ★★★ | フロストパイク / オーロレル |
| 雷雲のテンペスト | ★★★★ | ヴォルトツナ / サンダージョー |
| 銀河の最果て | ☆ NUSHI | **伝説のヌシ・レヴィアコス** |

進行状況・釣果図鑑はブラウザの localStorage に自動保存されます。

## 構成

```
space-fishing/
├── index.html
├── css/style.css
└── js/
    ├── utils.js      共通ユーティリティ＆擬似3D投影
    ├── input.js      タッチ／キーボード／ゲームパッド統合入力
    ├── audio.js      WebAudio 合成BGM＆効果音
    ├── fish.js       宇宙魚データ＆描画
    ├── planets.js    惑星データ＆背景（星空・星雲）
    └── game.js       メインループ／状態機械／釣りファイト
```

グラフィック・効果音・BGM はすべてコード内で手続き生成しており、**外部アセットには一切依存していません**。
